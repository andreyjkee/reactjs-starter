# reactjs-starter
[![NPM](https://nodei.co/npm/reactjs-starter.png?downloads=true&downloadRank=true&stars=true)](https://npmjs.org/reactjs-starter)
# Description
reactjs-starter is simple starter kit for start development with reactjs.
# How to use:
For production build run:
```bash
npm run build
```
For webpack-dev-server with HMR run:
```bash
npm run server:dev
```
## Libs:
 - `react`
 - `react-router`
